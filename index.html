IM Leagues Usability Test

Login to IM Leagues

Time to complete: 1 minute

The user was unsuccessful because they did not have an account. Besides this, the login page was intuitive. The page was on the slower side when loading content.

The site has a useful autocomplete feature when selecting the school you belong to.

Find and list active IM leagues for Fall '22

Time to complete: 40 seconds

The user took 5 extra clicks to find the current list of active leagues. We think this is because of the indirect step of first going to Davidson's page to find current leagues, instead of stepping directly to the leagues page from your profile.

I recommend increasing the size and improving the location of the Davidson college button.

Find the name of the Fall '22 flickerball team from your freshman year hall

Time to complete: 40 seconds

The user took just 1 extra click to reach the target. The flow is intuitive and easy to follow. The user was only tripped up when he clicked on the “Teams” tab on the nav bar and nothing changed. They were already on the “Teams” page, there was just no indication from the UI.

I recommend keeping the black bar underneath the “Teams” tab of the nav bar while the user is still there.

Find the name of the Fall '22 flickerball team from your freshman year hall

Time to complete: 1 minute

The user took a different path to the target than we anticipated when designing the test. Multiple routes to the same page helps accomodate users who do not follow the same path but have the same end goal.

I recommend a responsive nav bar design that expands this list when there is so much whitespace available.

Inspect element and find images with “alt” text

Time to complete: 1 minute 20 seconds

The user found two images with “alt” text: the university logo and the user profile picture. This is a good indication that a screen reader would be able to successfully and meaningfully describe pictures to users.

The site manages image accessibility well, allowing users who need screen readers easy usability.
